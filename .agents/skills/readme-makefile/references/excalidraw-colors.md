# Excalidraw Color Palette

## Semantic Colors

| Component | strokeColor | backgroundColor | Use Case |
|-----------|-------------|------------------|----------|
| Frontend/User | `#0891b2` (cyan) | `#cffafe` | UI, user-facing elements |
| Backend/Service | `#1e40af` (blue) | `#dbeafe` | API, services, processing |
| Database | `#7c3aed` (purple) | `#ede9fe` | Data storage, persistence |
| External/Cloud | `#ea580c` (orange) | `#ffedd5` | Third-party, external APIs |
| Start/Input | `#059669` (green) | `#d1fae5` | Entry points, origins |
| End/Output | `#dc2626` (red) | `#fee2e2` | Endpoints, results |
| Queue/Async | `#ca8a04` (yellow) | `#fef9c3` | Message queues, async ops |
| Cache | `#06b6d4` (teal) | `#cffafe` | Caching layers |

Glossary
========

This section contains definitions of terms used in wpipe.

API
~~~

Application Programming Interface. A set of protocols and tools for building software applications.

Decorator
~~~~~~~~~

A function that wraps another function to extend its behavior.

Health Check
~~~~~~~~~~~

A periodic check to verify that a worker is still responsive.

Nested Pipeline
~~~~~~~~~~~~~~

A pipeline that includes another pipeline as one of its steps.

Pipeline
~~~~~~~~

A sequence of steps that process data and produce results.

Step
~~~~~

A single function or callable that processes data in a pipeline.

Task
~~~~

A unit of work in a pipeline.

Worker
~~~~~~

A registered instance that executes pipeline processes.

Worker ID
~~~~~~~~~

A unique identifier for a registered worker.

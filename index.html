Simplicity First

Zero Boilerplate,
Pure Python

Define your steps, set your data flow, and let wpipe handle the orchestration, logging, and error recovery.

  • Automatic context passing between steps
  • Native async support for high-concurrency
  • Built-in resource monitoring
from wpipe import Pipeline

# Simple, modular steps
def fetch(data): return {"raw": "data"}
def process(data): return {"ok": True}

# Orchestrate in seconds
pipe = Pipeline(verbose=True)
pipe.set_steps([
    (fetch, "FetchData", "v1.0"),
    (process, "Transformer", "v1.0")
])

result = pipe.run({})
